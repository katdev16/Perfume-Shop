E-Commerce Project README

E-Commerce Project

Welcome to the E-Commerce project! This project is a fully functional online store built using React for the frontend and Spring Boot for the backend.

Features

  • User authentication (Sign up, Login, Logout)
  • Product listing and filtering
  • Shopping cart functionality
  • Secure checkout and payment integration
  • Order management and tracking

Installation

To set up the project locally, follow these steps:

Backend (Spring Boot)

  1. Clone the repository: git clone https://github.com/yourusername/ecommerce-backend.git
  2. Navigate to the backend directory: cd ecommerce-backend
  3. Install dependencies: mvn clean install
  4. Run the application: mvn spring-boot:run

Frontend (React)

  1. Clone the repository: git clone https://github.com/yourusername/ecommerce-frontend.git
  2. Navigate to the frontend directory: cd ecommerce-frontend
  3. Install dependencies: npm install
  4. Start the development server: npm start

Technologies Used

  • Frontend: React, Redux, Tailwind CSS
  • Backend: Spring Boot, Java, Spring Security
  • Database: MySQL
  • Payment Integration: Stripe
